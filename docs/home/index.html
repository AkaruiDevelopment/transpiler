Aoi-Transpiler

A Transpiler that converts aoi.js code to javascript.


Example


  const {
  Transpiler
  } = require( 'aoi-transpiler' );

  const code = `
  $let[hi;1]
  $get[hi]
  `;

  const func = Transpiler( code, true, {}, true, ).func;

  console.log( func );
